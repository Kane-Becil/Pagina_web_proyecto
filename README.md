# Proyecto: Sitio Web Personal de Ingeniería Informática

## Descripción del Trabajo
Este proyecto consiste en la creación de un sitio web personal para documentar y mostrar todo lo aprendido
durante la asignatura. El sitio incluye varias secciones:

- **Index.html:** Página principal con información general.
- **About.html:** Información personal, CV, imágenes y enlaces a redes sociales.
- **Degree.html:** Descripción detallada del grado en Ingeniería Informática.
- **Fii.html:** Página dedicada a la asignatura Fundamentos de la Ingeniería Informática.
- **Contact.html:** Formulario de contacto sencillo.
- **Net.html:** Enlaces a sitios web de compañeros.
- **Topic.html:** Tema libre, en este caso sobre videojuegos.

El sitio está diseñado usando **HTML5** y **CSS3** y sigue una estructura clara y coherente con carpetas
separadas para CSS e imágenes.

## Problemas Durante el Desarrollo
- Inicialmente la ruta de las imágenes causó problemas por nombres de carpetas incorrectos.
- Hubo errores de enlace entre `degree.html` y `fii.html` debido a mayúsculas y minúsculas.
- Configuración inicial de Git y GitHub Pages que requirió ajustar la carpeta del proyecto y los commits.

## Conclusiones
- Se logró implementar un sitio web funcional y visualmente atractivo.
- Se adquirió experiencia práctica en HTML, CSS, organización de proyectos web y control de versiones con Git.
- El proyecto permite documentar habilidades y presentar el trabajo de manera profesional.

## Información Adicional
- Todas las imágenes se almacenan en la carpeta `images`.
- Todos los estilos CSS se centralizan en `css/styles.css`.
- El repositorio de GitHub es público y actualizado constantemente con cada cambio.
